## 구현된 클래스

### ConfigDiscoveryModule
- `RobotsTxtParser` - robots.txt 파싱 (Disallow, Sitemap)
- `SitemapParser` - sitemap.xml 파싱 (nested sitemap 지원)
- `WellKnownParser` - security.txt, openid-configuration 파싱
- `SourceMapDetector` - JavaScript source map 탐지

### HtmlElementParserModule
- `FormParser` - form action, method, hidden inputs, CSRF 토큰
- `ScriptParser` - script src, inline URL 추출
- `LinkParser` - link href, rel types
- `MediaParser` - img, video, audio, source, srcset
- `MetaParser` - meta refresh, Open Graph, Twitter Card
- `DataAttrParser` - data-* 속성 URL

### ResponseAnalyzerModule
- `HeaderAnalyzer` - Server, X-Powered-By, CORS
- `CspParser` - Content-Security-Policy 도메인 추출
- `CookieAnalyzer` - Set-Cookie 분석 (도메인, 경로, 보안 플래그)
- `ErrorDetector` - 에러 메시지 경로 노출, 스택 트레이스 탐지
